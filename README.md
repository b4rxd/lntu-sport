# LNTU CRM application

## Local deploy

### Download and install

```shell
git clone https://github.com/b4rxd/lntu-sport.git
cd lntu-sport
composer install
npm i
php artisan migrate --seed
```

### Run local

Then run in separate terminals
```shell
npm run dev
php artisan serve
```
